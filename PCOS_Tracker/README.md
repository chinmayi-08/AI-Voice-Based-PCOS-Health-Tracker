# AI PCOS Health Tracker

A simple healthcare-focused desktop application built using Python and Tkinter to help users track PCOS-related health information such as symptoms, sleep, mood, water intake, and menstrual cycle details.

---

## Features

- Track daily symptoms
- Monitor water intake
- Record sleep hours
- Mood tracking
- Period date tracking
- Save health records using CSV
- View previous records
- Simple and user-friendly GUI
- Desktop executable (.exe) support

---

## Technologies Used

- Python
- Tkinter
- CSV File Handling
- PyInstaller
